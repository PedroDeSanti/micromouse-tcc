## Visualização 3D

O projeto mecânico do Micras v2 foi guiado por uma filosofia de Co-Design, onde a estrutura física e a eletrônica foram desenvolvidas simultaneamente para atingir um footprint ultracompacto de aproximadamente 50×90 mm.

Adotando a arquitetura PCB-as-Chassis, a própria placa de circuito atua como o elemento estrutural principal, eliminando o peso morto de frames adicionais. Abaixo, você pode explorar o modelo 3D interativo para visualizar os detalhes dessa montagem.
