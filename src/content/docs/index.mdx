## Sobre o Projeto

Este projeto apresenta o reprojeto completo de uma plataforma autônoma para a categoria Micromouse. Nascido da necessidade de superar limitações mecânicas e computacionais, o **Micras v2** integra uma nova arquitetura de tração diferencial com sucção (Downforce) e um sistema embarcado de alta densidade baseado no **STM32H7** (Cortex-M7 de 550 MHz).

Com uma PCB de 4 camadas projetada para integridade de sinal e um firmware modular em C++ capaz de executar algoritmos avançados de planejamento de trajetória e controle Feed-Forward, o robô não apenas navega, mas "resolve" labirintos com inteligência e velocidade extremas. Todo o projeto é **Open Source**, visando impulsionar a comunidade brasileira de robótica.
